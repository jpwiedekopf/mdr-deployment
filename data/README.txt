MDR Exports
-----------

This directory contains various MDR exports, that document the
incorporation of BBMRI Locator attributes into the GBA biobank data
model.

gba_sample_locator_mdr16.xml	Original mdr16 export, made by Noemi
bbmri_eric_locator_mdr16.xml	Export incorporating changes for BBMRI-ERIC European Locator
ejprd_sample_locator_mdr16.xml	Export incorporating additional rare disease attributes
